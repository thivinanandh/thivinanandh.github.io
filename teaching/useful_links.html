Useful Learning Resources

Here are some useful links that I have found helpful in my research and teaching.

C++
Understanding Pointers in C++ (MycodeSchool)
Excellent playlist on understanding Pointers in C++ (The author has good playlist on understanding DSA)
The Cherno Channel (The Cherno)
A very good channel to get familiarised understanding C++ Syntaxes.
FEM (Finite Element Methods)
FEM Course (openmichigan)
A very good introductory course on Finite Element methods by Prof.Krishna Garikapathi. Covers both theoretical and practical aspects.
FEM Mathematics Guide
An excellent material for understanding the Math and Matrices behind Finite Element Methods by Prof. CS Jog at IISc
FEM Algorithms: Implementation Perspective
Chapter-9: Great material on Object Oriented Implementation of Finite Element Softwares by Prof. Sashi
Numerical Methods
Linear Algebra and PDEs (MathTheBeautiful)
The author is PhD Student of Gilbert Strang. Has very good lectures on intersection of Linear Algebra and its association with numerically solving PDE's
Scientific Machine Learning
DeepONets (JHU)
Presentation on Neural Operators(DeepONets) by Prof. George Karniadakis
Crunch Group Seminars (Crunch-Group)
A series of weekly seminars on latest advances in Scientific Machine Learning.
SciML Lectures (CAMLab, ETH Zurich)
Lectures on SciML concepts such as PINNs, Neural Operators, Symbolic Regression etc offered at ETH Zurich by Dr. Ben Moseley and Prof. Siddhartha Mishra
Machine Learning
Math in ML
Mathematics for Machine Learning
Great book on understanding the mathematics behind ML Algorithms
Neural Networks Visual Guide (3Blue1Brown)
Visual explanation of Neural Networks and Transformer Architectures
Linear Algebra Essentials (3Blue1Brown)
Visual explanation of Linear Algebra concepts
ML Algorithms
StatQuest (Statquest)
Excellent channel explaining all ML Algorithms (Good for revision)
ML Libraries
Tensorflow-Tutorials (mrdbourke)
Excellent collection of notebooks for learning tensorflow. Good learning material for the Google's Tensorflow developer certification
ML Projects & Coding
500 ML/DL Projects
A collection of ML & DL Projects
MLOps
Data Version Control (DVCorg)
Explaining Data version control
Other Coding
The Coding Train (TheCodingTrain)
Excellent resource on coding physics based animations. Also Refer to "The Nature of Code"
Kevin Fang (Kevin Fang)
A great analysis on production level bugs
Stand-up Maths (standupmath)
A casual study on random math problems in real life. Matt Parker is one of my Favourite Authors. His book "Humble Pi", A book about what happens wrong when math goes wrong in real life is must read. If you are not patient to read, watch this video. It has a short summary of what is discussed in the book.
Fireship (Fireship)
Good channel for getting tech updates
Practical Engineering (Practical Engineering)
Visual demonstration of Fluid Mechanics principles
